# 文档站说明

这里是楸晟的个人文档站，用来保存可复用、可检索、可长期维护的内容。

## 内容范围

- 技术学习笔记
- 项目背景、方案和复盘
- 部署、运维和工具链记录
- 常用命令、配置和排障经验

## 写作约定

每篇文档尽量说明三个问题：

1. 这件事解决什么问题。
2. 当前采用什么做法。
3. 后续维护时需要注意什么。

这样以后回来看，不只是看到命令和配置，也能理解当时为什么这样做。

## 当前状态

站点已经从 VitePress 官方文档模板切换为个人文档站结构。后续可以按项目、主题或时间继续扩展目录。
